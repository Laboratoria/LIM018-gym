# CSS Lock Up

[https://cssbattle.dev/play/27](https://cssbattle.dev/play/27)

__Objetivo__

Escreva o HTML/CSS no editor para replicar a imagem de destino à direita.

__Código de base__

```html
    <div></div>
    <style>
      div {
        width: 100px;
        height: 100px;
        background: #dd6b4d;
      }
    </style>
```
